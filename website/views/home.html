Welcome!

Welcome to my Data Science project on gun violence in America. This website explores the scope, patterns, and trends of firearm-related incidents across the country using real datasets, visualizations, and machine learning models.

Project Overview

The project covers:

  • Data collection, cleaning, and exploratory analysis (EDA)
  • Clustering and PCA to find patterns
  • Supervised learning models including Naive Bayes, Decision Trees, SVM, Regression, and Neural Networks
  • Visualization of results and insights
  • Non-technical conclusions for general audiences
